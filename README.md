# InboxIQ: Agentic Email Assistant

## Project Overview

InboxIQ is an agentic AI system built with IBM watsonx Orchestrate that triages emails, classifies urgency, and takes automated action through Slack and calendar integration.

### Problem

Knowledge workers waste hours managing email interruptions.

### Solution

AI agent decides:

Urgent → Slack

Actionable → Calendar

Informational → No interruption
